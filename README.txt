BookNest 配布一式
=================

内容
----
index.html                 アプリ本体
manifest.webmanifest       PWA設定
service-worker.js          アプリ画面のキャッシュ
favicon.ico                ブラウザ用アイコン
icons/                     SVGおよび各サイズのPNGアイコン

公開方法
--------
このフォルダの中身を、同じ階層構造のままHTTPS対応のWebサーバーへアップロードしてください。
GitHub Pagesでも利用できます。カメラとService Workerは file:// で直接開いた場合には動作しません。

iPhoneでホーム画面へ追加
------------------------
1. Safariで公開URLを開く。
2. 共有ボタンをタップする。
3. 「ホーム画面に追加」を選ぶ。
4. 名前が BookNest になっていることを確認して追加する。
5. カメラ利用を求められたら許可する。

iPhoneでは manifest より apple-touch-icon が優先されるため、icons/booknest-180.png を用意しています。

Androidでホーム画面へ追加
--------------------------
Chromeで公開URLを開き、メニューから「ホーム画面に追加」または「アプリをインストール」を選びます。

データについて
--------------
蔵書データは従来どおりブラウザのlocalStorageに保存されます。既存版と同じ保存キーを維持しているため、
同じ公開URLでindex.htmlを置き換えれば、従来の蔵書データをそのまま読み込めます。
端末変更やブラウザデータ削除に備えて、設定からJSONバックアップを定期的に保存してください。

オフライン動作
--------------
一度オンラインで開いた後は、アプリ画面と読み込み済みのスキャナライブラリをキャッシュします。
ただし、openBD・NDL Search・Open Library・Google Booksを使う書誌情報や表紙の検索には通信が必要です。
